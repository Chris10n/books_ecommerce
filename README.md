# Books eCommerce Platform

A simple online bookstore application built with PHP, JavaScript, and MySQL. Features product browsing, shopping cart management, and order checkout.

## Tech Stack

| Category | Technology |
|----------|-----------|
| Frontend | HTML5, CSS3, JavaScript |
| Backend | PHP 8.2+ |
| Database | MySQL / MariaDB |
| Server | Apache (XAMPP) |

## Features

- Product browsing with search functionality
- Shopping cart management (add, remove, update quantities)
- Checkout with customer information form
- Order processing and confirmation
- Session-based cart persistence
- Responsive design
