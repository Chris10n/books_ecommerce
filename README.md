# Books eCommerce Platform

A simple online bookstore application built with PHP, JavaScript, and MySQL. Features product browsing, shopping cart management, and order checkout.

## Tech Stack

| Category | Technology |
|----------|-----------|
| Frontend | HTML5, CSS3, JavaScript |
| Backend | PHP 8.2+ |
| Database | MySQL / MariaDB |
| Server | Apache (XAMPP) |

## Features

- Product browsing with search functionality
- Shopping cart management (add, remove, update quantities)
- Checkout with customer information form
- Order processing and confirmation
- Session-based cart persistence
- Responsive design

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Chris10n/books_ecommerce.git
   cd books_ecommerce
   ```

2. **Move to XAMPP htdocs folder**
   ```
   Copy to: C:\xampp\htdocs\books_ecommerce\
   ```

3. **Start XAMPP** - Open XAMPP Control Panel and start Apache & MySQL

4. **Create Database**
   - Open phpMyAdmin: `http://localhost/phpmyadmin`
   - Create database: `books_db`
   - Import `books_db.sql` file

5. **Run the application**
   ```
   http://localhost/books_ecommerce/
   ```
